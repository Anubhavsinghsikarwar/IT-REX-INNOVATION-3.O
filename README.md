# IT-REX-INNOVATION-3.O
it about cab sharing program where we book a cab or auto through sharing which leads to cost effective for users.
1. User visits localhost
2. User sees a basic UI with a destination input field
3. User submits the destination
4. Backend receives the destination
5. Backend fetches ride prices from ride apps (Uber, Rapido, etc.)
6. Backend extracts and normalizes the data
7. Prices are displayed on the UI:
   - Bike
   - Auto
   - Cab
8. User selects an option
9. Based on the choice:
   - Generate a chat room ID
   - Or automatically pair users going to the same destination
10. Final confirmation is shown to the user
